# Notes — GraphQL-сервер (Bun + TypeScript + MongoDB)

### Быстрый старт
1) Клонируйте репозиторий и перейдите в папку проекта
```bash
git clone <URL_РЕПОЗИТОРИЯ> notes
cd notes
```

2) Установите зависимости
```bash
bun install
```

3) Запустите сервер
```bash
bun index.ts
```

Откройте `http://localhost:4000/graphql`.
